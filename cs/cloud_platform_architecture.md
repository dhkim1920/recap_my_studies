# 플랫폼 아키텍처

## 하이브리드 멀티 클라우드 환경

- **하이브리드 클라우드**: 온프레미스(자체 데이터센터)와 퍼블릭 클라우드(AWS, Azure 등)를 병행 운영  
  - 민감 데이터는 온프레미스에, 나머지는 퍼블릭 클라우드에 저장

- **멀티 클라우드**: 여러 퍼블릭 클라우드(AWS, GCP, Azure 등)를 동시에 사용  
  - 벤더 종속 최소화, 장애 대응력 강화

- **하이브리드 멀티 클라우드**: 온프레미스 + 다수 클라우드를 조합한 복합 구조  
  - 데이터 및 워크로드를 특성과 비용에 맞게 분산 배치
