# VSCode 관련 정리

## Extensions

### 1. Git 및 테마

- **GitLens**: 코드 라인별로 작성자, 수정 시간을 실시간으로 보여줌
- **Material Icon Theme**: 파일 및 폴더 아이콘을 직관적인 디자인으로 변경 (기본 아이콘이 시인성이 떨어짐)

### 2. 프로그래밍 언어

- **Python (Microsoft)**: IntelliSense, 디버깅, 린팅, 코드 포맷팅 등 파이썬 개발에 필요한 기능을 통합 제공

### 3. 컨테이너 및 인프라

- **Docker (Container Tools)**: 컨테이너 관리, 이미지 빌드, 배포를 지원
- **Kubernetes**: 클러스터 상태 확인 및 리소스 매니페스트 작성을 지원

## 주요 Shortcuts

| 단축키 | 기능 설명 |
| --- | --- |
| F12 | **Go to Definition**: 함수나 변수가 선언된 위치로 이동 |
| Ctrl + P | **Quick Open**: 파일명 검색 및 이동 |
| Ctrl + F | **Find**: 현재 파일 내 검색 |
| Ctrl + Shift + F | **Global Search**: 프로젝트 전체 검색 |
| Ctrl + Shift + P | **Command Palette**: 모든 명령 실행 및 검색 |
| Ctrl + \ | **Split Editor**: 편집기 화면 분할 |
| Ctrl + Shift + V | **Privew**: 화면 미리보기 |
| Ctrl + B | **Toggle Side Bar**: 사이드바 표시/숨김 |
| Alt + Up/Down | **Move Line**: 줄 바꿈 (코드 이동) |
